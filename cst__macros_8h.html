Detailed Description

Macros for accessing fields of node structs.

rdesc returns opaque pointers abstracting away underlying complexity of memory layout. This header provides stable macros for dereferencing such pointers to terminal/nonterminal fields that are tentative and unstructured due to memory layout optimizations.
