Detailed Description

Stack interface contract.

This header defines the function signatures required to manipulate the parser's backtracking stack.

Note
For custom implementations: If you have defined a custom struct rdesc_stack, you must provide implementations for all the functions declared in this file. The parser engine relies on these exact signatures to perform backtracking.
