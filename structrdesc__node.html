Detailed Description

A node in the CST.

Uses a union to allow a single node type to represent both (tokens and non-terminals. The ty field in the union determines which interpretation to use (0 = token, 1 = nonterminal).

The flexible array member _[] allows extending seminfo storage beyond the base token size when seminfo_size > RDESC_BASE_SEMINFO_SIZE.
